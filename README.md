# 🎓 Campus Connect

> An AI-powered academic resource-sharing platform for students 

Campus Connect eliminates "resource hunting" by combining smart recommendations, 
Gemini AI, and faculty-verified content — so students spend 100% of their time 
studying, not searching.

## ✨ Key Highlights
- 🤖 **AI Study Tools** — Syllabus extraction, PYQ exam predictor & PDF summarization via Gemini Pro
- 📈 **Smart Feed** — Personalized dashboard that learns from your search behavior
- 🏅 **Faculty Verification** — Gold trust badges on professor-vetted resources
- 🔥 **Trending Engine** — Live polling + algorithmic ranking (Likes × 3 + Downloads × 2)
- 🎨 **Modern UI** — Glassmorphism, Framer Motion micro-animations & skeleton loaders

## 🛠 Tech Stack
Next.js 14 · PostgreSQL · Prisma · Supabase Auth · AWS S3 · Gemini Pro · Tailwind CSS · Vercel
